# Random Tables

## <dfn id = "term-random-element-table">Random Element Table</dfn>

When a [character](/srd/glossary#term-character) uses an [item](/srd/economy/items), [ability](/srd/glossary#term-ability), or something else that has a random [elemental](/srd/combat/elements#term-element) effect, cast the [outcome die](/srd/glossary#term-outcome-die) and consult the table below to determine the result.  If result is a 10, cast again instead.

|Result|Element|
|:-|:-|
|1|[Fire](/srd/combat/elements#term-fire)|
|2|[Water](/srd/combat/elements#term-water)|
|3|[Earth](/srd/combat/elements#term-earth)|
|4|[Wind](/srd/combat/elements#term-wind)|
|5|[Thunder](/srd/combat/elements#term-thunder)|
|6|[Ice](/srd/combat/elements#term-ice)|
|7|[Wood](/srd/combat/elements#term-wood)|
|8|[Light](/srd/combat/elements#term-light)|
|9|[Shadow](/srd/combat/elements#term-shadow)|
|10|Recast|

## <dfn id = "term-random-statistic-table">Random Statistic Table</dfn>

When a character uses an item, ability, or something else that affects a random [statistic](/srd/heroes/numbers#term-statistic), cast the outcome die and consult the table below to determine the result.  If the result is a 10, cast again instead.

|Result|Statistic|
|:-|:-|
|1|[Physical Attack](/srd/heroes/numbers#term-strike)|
|2|[Physical Defense](/srd/heroes/numbers#term-physical-defense)|
|3|[Physical Accuracy](/srd/heroes/numbers#term-physical-accuracy)|
|4|[Magical Attack](/srd/heroes/numbers#term-spell)|
|5|[Magical Defense](/srd/heroes/numbers#term-magical-defense)|
|6|[Magical Accuracy](/srd/heroes/numbers#term-magical-accuracy)|
|7|[Evasion](/srd/heroes/numbers#term-evasion)|
|8|[Resistance](/srd/heroes/numbers#term-resistance)|
|9|[Status Accuracy](/srd/heroes/numbers#term-status-accuracy)|
|10|Recast|
