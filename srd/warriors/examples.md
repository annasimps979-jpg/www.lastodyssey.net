*The only proof of a warrior is combat.*

One hundred example warriors are provided, ten per [tier](/srd/heroes/creation#term-tier).
Each tier’s warriors are divided into two thematic groups of five; each theme relates them to example
[lore](/srd/discovery/lore).

All example warriors were composed using the [Warrior Creation](/srd/warriors/creation) tools.
