#  Items

*Good in a pinch, even better in a firm grip.*

An _<dfn id = "term-item">item</dfn>_ is any object in a [character's](/srd/key-terms#term-character) [inventory](/srd/economy/inventory#term-inventory) which, when used, can produce a single, discrete effect.  After an item is used, it is expended and leaves its user's inventory.

- There Are Two Varieties
    - [Battle Items](/srd/economy/items/battle): Items whose primary usage is during combat.
    - [Travel Items](/srd/economy/items/travel): Items whose primary usage is during exploration.
